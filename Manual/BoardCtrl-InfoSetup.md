### Game Information Setup
By clicking icons or buttons of 'New File', 'Add Game', 'Edit Information' and 'Game Setup', it shows a dialog shown below. It could be little different case by case but mostly identical in detail.

<img src="img/Board 8 - Add game.png"  width="60%" alt="Game setup overall"/>      

The game setup dialog provides information setup such as game title and it description, board size to play and the rules to apply, players to run the game even with the popular Go engines in the market, and more.
