### Information and View Control
<img src="img/Board 1 - Board only.png"  width="60%" alt="Board only overall"/>         
As shown, the main view of the board is divided to 3 parts as below.

|            |                   |  
|:-----------|:------------------|
|<img src="img/Board 1 - Game Info.png" width="60%" alt="Game info"/>       |'Game information' panel at left shows game setup values.<br>Game title and its short description<br>Web link if defined<br>Player information and applied game rule<br>Game edit button if the information is not locked.   |              
|<img src="img/Board 1 - Mode control.png" width="60%" alt="Mode control"/> |'Mode and display control' at right shows the current selected board mode and the controls that override or detail control determined from the status bar.<br>Shows check buttons for stone number and the last stone display.<br>shows check buttons to set the mouse action on the board.<br><br>Shows 'Game Analysis' buttons that gives the analysis result for territory by GNU Go or move recommendation by Leela Zero LITE version<br><br>'Board Message' at the bottom displays what and how the board is doing. If something needs to caution, warning sound with red letter is displayed.|
