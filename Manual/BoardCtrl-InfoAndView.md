### Information and View Control
Main window of the app

<img src="img/Board 1 - Board only.png"  width="60%" alt="Board only overall"/>   

As shown, the main view of the board is divided to 3 parts as below.

|            |                   |  
|:-----------|:------------------|
|<img src="img/Board 1 - Game Info.png" width="60%" alt="Game info"/>       |'Game information' panel at left shows game setup values.<br>Game title and its short description<br>Web link if defined<br>Player information and applied game rule<br>Game edit button if the information is not locked.   |              
|<img src="img/Board 1 - Mode control.png" width="60%" alt="Mode control"/> |'Mode and display control' panel at right shows the current selected board mode and the controls that override or detail control determined from the status bar.<br>Shows check buttons for stone number and the last stone display.<br>shows check buttons to set the mouse action on the board.<br><br>Shows 'Game Analysis' buttons that gives the analysis result for territory by GNU Go or move recommendation by Leela Zero LITE version<br><br>'Board Message' at the bottom displays what and how the board is doing. If something needs to caution, warning sound with red letter is displayed.|
|<img src="img/Board 1 - Board center.png" alt="Board center"/> |Board shows the determined size of board and the stones if exists.|  

Board controls at the bottom

<img src="img/Board 1 - Board controls.png" alt="Board controls"/>

|            |                   |  
|:-----------|:------------------|
|<img src="img/Board 1 - Jump and Autoplay.png" alt="Jump and Autoplay"/>       |'Jump and AutoPlay' put the stone number into the box and click or enter displays the stone list to the number. Or, AutoPlay button plays the stone list from the current view to the end or until stop button clicked. |
|<img src="img/Board 4 - Auto time interval.png" alt="Autoplay time interval"/> |'TIme Interval' setup is available for AutoPlay    | 
|<img src="img/Board 1 - Traverse.png" alt="Traverse button"/>                  |'Traverse' buttons. |
|<img src="img/Board 1 - Board status.png" alt="Board status"/>                 |'Board status' display. The display shows currently 19x19 size of board, cusor on board is empty large size square. The last button clicks enlarge the board display by removing the status bar that can be useful on small size of display.| 
|<img src="img/Board 5 - Display control.png" alt="Board display"/>             |Move the cursor on the board and click the right button of mouse. It shows current board setting values as shown.<br>User can select graphic board or monochrome board. Also able to show or hide controls of the board      |
|<img src="img/Board 6 - Background setup.png" alt="Board background"/>         |Move the cursor on the left or right side of the board and click the right button of mouse. It shows also the menu bar that user can determine the display of the board background.                                            |
