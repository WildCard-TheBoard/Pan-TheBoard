### Board Status Control
Second row of the app

<img src="img/Board 1 - Status bar.png"  width="100%" alt="Board status bar overall"/>     

|            |                   |  
|:-----------|:------------------|
|<img src="img/Board 1 Add game.png" alt="Add game"/>              |'InfoCenter' opens this information website.<br> 'Add Game' opens new game setup dialog.<br>'Import' opens File open dialog to add a game list to the existing game file.<br>'Export' opens File save dialog to save a game file from the currently opened game view.|
|<img src="img/Board 1 Tools.png" alt="Board tools"/>              |'Player' shows the player to play now. Currently 'White' is the turn to move in the figure.<br>User is able to change the player by clicking this icon.<br> Also, user can change the existing stone color by cliking this icon when a stone is selected. This will be happen when the change does not break the Go rule.<br>'Pen' opens pen tools to free draw on the board.<br>'Scenario' switches the currently opened stone layout from reguar game mode to scenario, or vice versa. <br>'InfoView' switches to show or hide a seperate window of game tree view.|
|<img src="img/Board 1 Display control.png" alt="Display control"/>|'Number' controls to display the stone number on the board.<br>'Marker' controls to show or hide marker on the board if exists.<br>'M.Add' turns on and off the marker add tool on the board.<br>'M.Inhrt' determines to keep or not the existing marker list to the next stone move.<br>'V.Mirror' mirrors the board vertically.<br>'H.Mirror' mirrors the board horizontally.<br>'Rotation' rotates the board 90 degree counterclockwise by a click. |
